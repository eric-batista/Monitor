Este projeto foi desenvolvido com as seguintes tecnologias:

* **Back-end:** Nest.js
* **Front-end:** Next.js
* **E-mail microservice:** Go
* **Apache Kafka**
* **Kafka Connect**
* **Docker e Kubernetes**
* **Confluent Cloud**
* **Google Cloud Platform**
